# Sample Medical Images for AIMS VISION PRO Demo

Drop real medical images here for live demo. Supported formats: JPG, PNG, DICOM (.dcm)

## Recommended images to add:
1. **prescription.jpg** — A handwritten or printed referral/prescription form
2. **knee_mri_01.jpg** — Knee MRI scan (sagittal view)
3. **knee_mri_02.jpg** — Knee MRI scan (coronal view)  
4. **chest_xray_01.jpg** — PA Chest X-ray
5. **chest_xray_02.jpg** — Lateral Chest X-ray

## Where to get sample images:
- Open source medical image datasets (e.g., OpenI, MIMIC-CXR)
- Place them in this folder and the AIMS upload flow will use them
- For demo: use the built-in mock cases at /api/demo/run

## Current mock cases available:
- knee_mri: Right Knee — Medial meniscus tear (3 images)
- chest_xray: Chest — Community-acquired pneumonia (2 images)
